Для постройки крепкого корабля, нужно сделать так, что бы в нем не было дырок. Для этого нужно сделать все доски нужного размера.

У вас есть магическая дерево размера K. Вам нужно разделить дерево на множество досок размер которых, будет простым числом. Так как дерево магическое, то его общая длина достигается не суммированием размера досок, а их перемножением.

Выведите множество досок отсортированных по возрастанию, которое может получиться из текущего размера дерева.

Например:
66 - размер магического дерева

То множество досок будет:
2 3 11

Входные данные:
0 < N <= 1e9 - размер магического дерева

Выходные данные:
A[0] A[1] ... A[A.size() - 1] - множество отсортированных по возрастанию досок, выведеное через пробел
