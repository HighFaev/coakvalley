"Символом появления надежды на спасение будет звездопад на небе..."

На листе с предсказанием четко указана последовательность из N звезд, которые должны устроить звездопад.

Для каждой звезды дано время S, когда звезда появилась на небе. Никаких две звезды не появились в одно и тоже время. Так же дано время F, когда звезда "упала". И для каждой звезды есть неизвестная величина D - длительность падения звезды.

Известно, что звезды падали в том порядке, в котором появлялись. Но звезды падали следующим образом:
1. Как только появилась новая звезда, она сразу начинает падать.
2. Если появилась новая звезда, пока падает еще предыдущая, то она становиться в конец очереди.
3. Когда звезда "упала" и очередь была не пуста, сразу начинает падать первая в очереди звезда. Если очередь пуста, то надо ждать появление новой звезды.

Найдите D(время падения звезды) для каждой звезды.

Например:

Входные данные:
1 ≤ T ≤ 1e3 - количество наборов входных данных в тесте
Далее следуют описания наборов входных данных.
В первой строке каждого набора входных данных содержится одно целое число N (1 ≤ n ≤ 1e2).
Во второй строке каждого набора входных данных записано ровно N целых чисел S1 < S2 < ⋯ < Sn (0 ≤ Si ≤ 1e5).
В третьей строке каждого набора входных данных записано ровно N целых чисел F1 < F2 < ⋯ < Fn (Si < fi ≤ 1e5).

Гарантируется, что сумма n по всем наборам входных данных не превышает 1e5
